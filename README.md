## Dependecies
- node (ver 10.16.3, recommend using [nodenv](https://github.com/nodenv/nodenv))
- expo-cli (install: npm install -g expo-cli)

## How to ran
### Prepare (at once first)
```bash
nodenv install 10.16.3 # if use nodenv
npm install -g expo-cli
npm install
```

### Run
```
npm start
(and follow screen...)
```
